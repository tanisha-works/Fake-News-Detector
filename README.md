# Veritas AI: Multilingual Fake News Detector

Veritas AI is a professional machine learning project designed to verify the authenticity of news articles. In an era of rapid misinformation, this tool helps users identify potentially fake news in both **English and Hindi**.

## Features
* **Multilingual Support:** Handles both English and Hindi text seamlessly using Google Translation API.
* **Modern Dashboard:** A clean, professional UI built with Streamlit for a great user experience.
* **ML Power:** Uses advanced NLP (Natural Language Processing) and Scikit-Learn models to detect deceptive language patterns.
* **Algorithm Comparison:** Shows a visual breakdown of different model performances.

## 🛠️ Tech Stack
* **Language:** Python
* **ML Libraries:** Scikit-Learn, Pandas, NumPy
* **UI Framework:** Streamlit
* **Translation:** Deep Translator
* **Deployment:** Streamlit Cloud
